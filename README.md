# Projet INF301 en trinome - L2 Informatique UGA : APPS
# INF301 Project in collaboration with two teammates - L2 Computer Science at UGA: APPs



APP1 : Cryptographie et Communication Flux : Mise en œuvre d'algorithmes de chiffrement (substitution, décalage) pour sécuriser des flux de données. Gestion de la communication client-serveur interactive via des sockets.
- Note technique - sur l'APP1 : Cette application repose sur une architecture client-serveur. L'exécution complète nécessite une connexion active au serveur pédagogique AppoLab.

APP2 : Interpréteur Curiosity (Version Avancée) : Développement d'un moteur d'exécution basé sur une pile pour interpréter des scripts d'instructions robotiques. Ce module inclut des tests de performance et une gestion robuste des erreurs d'exécution.

APP3 : Arbres Phylogénétiques : Utilisation d'arbres binaires pour la classification taxonomique du vivant. Implémentation d'algorithmes récursifs pour la recherche, l'insertion d'espèces et l'analyse de caractéristiques biologiques.


## 


APP1: Cryptography & Stream Analysis: Implementation of encryption algorithms (sequence and move-based encryption) to secure data streams. Includes interactive client-server communication modules.
- Technical Note - on APP1: This application is based on a client-server architecture. Full execution requires an active connection to the AppoLab server.

APP2: Stack-based Interpreter (Curiosity): Design of a stack-driven execution engine for interpreting robotic instruction sets. Features robust error handling and performance evaluation scripts.

APP3: Phylogenetic Trees & Classification: Binary tree implementation for biological species classification. Includes recursive algorithms for tree traversal, species insertion, and trait analysis.
